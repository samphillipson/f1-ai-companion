# F1 AI Companion: Walkthrough

The F1 AI Companion project has been successfully initialized, built, and recently patched to ensure long-term stability and functional API integrations. Here is a summary of what the project contains and how to get started.

## What Was Built

- **Next.js Fullstack Architecture**: A secure, modern web application using React and Next.js App Router.
- **Premium UI Design**: A bespoke F1-inspired design system with dark mode, glassmorphism, dynamic animations, and responsive layouts.
- **Jolpica API Integration**: A robust utility `src/lib/jolpica.ts` to seamlessly fetch current and historical F1 data. (Recently patched to use `https://` for functional deployment).
- **Next Race Dashboard**: The dashboard automatically determines and displays the next upcoming F1 race from the schedule.
- **Context-Aware AI Chatbot**: An interactive AI assistant UI (`src/components/Chatbot.tsx`) connected to a secure backend API route (`/api/ai`). Before generating responses, it fetches the current Driver Standings and Next Race data and seamlessly injects it into the prompt so the AI possesses accurate, real-time knowledge instead of outdated training data.
- **Fail-Fast Rate Limit Handling**: The chatbot is configured to gracefully display API warnings directly in the UI if you hit the Free Tier rate limit or encounter Google API server congestion (`429` and `503` errors).

## API Keys & Setup

> [!IMPORTANT]
> You are currently using a Google Gemini API key on the Free Tier, which may occasionally trigger a `429 Too Many Requests` or `503 Service Unavailable` error due to high demand.
> 
> If you wish to upgrade or rotate your API key:
> 1. Go to **[Google AI Studio](https://aistudio.google.com/app/apikey)**
> 2. Sign in with your Google account.
> 3. Click **"Create API key"**.
> 4. Copy the generated key and paste it into the `.env.local` file located in your project root at `C:\Users\samph\.gemini\antigravity\scratch\f1-ai-app\.env.local`.

## How to Run the App

1. Open a terminal and navigate to the project:
   ```bash
   cd C:\Users\samph\.gemini\antigravity\scratch\f1-ai-app
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open your browser and navigate to `http://localhost:3000`.

The dashboard and AI chatbot are fully synced to GitHub and ready for your next coding session!
