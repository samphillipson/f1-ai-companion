# F1 AI Companion: Walkthrough

The F1 AI Companion project has been successfully initialized and built! Here is a summary of what was accomplished and how you can get started.

## What Was Built

- **Next.js Fullstack Architecture**: Created a secure, modern web application using React and Next.js App Router.
- **Premium UI Design**: Implemented a bespoke F1-inspired design system with dark mode, glassmorphism, dynamic animations, and responsive layouts.
- **Jolpica API Integration**: Built a robust utility `src/lib/jolpica.ts` to seamlessly fetch current and historical F1 data.
- **AI Chatbot**: Added an interactive AI assistant UI (`src/components/Chatbot.tsx`) connected to a secure backend API route (`/api/ai`), protecting your AI API keys.
- **NextAuth Security**: Pre-configured user authentication (`/api/auth/[...nextauth]`) to secure your application.

## API Keys & Setup

> [!IMPORTANT]
> You asked where to find your Google Gemini API key:
> 1. Go to **[Google AI Studio](https://aistudio.google.com/app/apikey)**
> 2. Sign in with your Google account.
> 3. Click **"Create API key"**.
> 4. Copy the generated key and paste it into the `.env.local` file located in your project root at `C:\Users\samph\.gemini\antigravity\scratch\f1-ai-app\.env.local`.

## How to Run the App

> [!NOTE]
> It appears Node.js is not installed locally on this machine. To run the app, you will need to install it first:
> 
> 1. Download and install Node.js from [nodejs.org](https://nodejs.org/).
> 2. Open a terminal and navigate to the project:
>    ```bash
>    cd C:\Users\samph\.gemini\antigravity\scratch\f1-ai-app
>    ```
> 3. Install the dependencies:
>    ```bash
>    npm install
>    ```
> 4. Start the development server:
>    ```bash
>    npm run dev
>    ```
> 5. Open your browser and navigate to `http://localhost:3000`.

The dashboard and AI chatbot are ready to go once the API key is in place and the server is running!
